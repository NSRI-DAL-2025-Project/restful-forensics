# restful-forensics

## Installation (installing without Docker)

### From the terminal
Clone the repository. Make sure that git is installed.
```
git clone https://github.com/NSRI-DAL-2025-Project/restful-forensics.git
```

### Using RStudio
- Under File > New Project > Version Control > Git
- Under Repository URL, paste: "https://github.com/NSRI-DAL-2025-Project/restful-forensics.git"

Open the "app.R" file and click the "Run App" button on the top right corner of the source panel.
![](www/readme/runapp.png)

Note: Some functionalities require a Linux OS. A Windows-compatible version will be linked here once available.
